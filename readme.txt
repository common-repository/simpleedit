== Description ==

After activate this Plugin you can make use of the Tab and Shift + Tab Key(s) in your WordPress Backend in the Html Editor. 
It is a nice feature to format Html code with tabulators while writing a Blog Post. <a href="http://wordpress.org/extend/plugins/simpleedit/screenshots/" target="_self" >Screenshot</a>
Especially if you want to post sample source code in different programming languages.

This section describes how to install the plugin and get it working.

   2. Activate the plugin through the ‘Plugins’ menu in WordPress
